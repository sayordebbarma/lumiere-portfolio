# Lumière

A cinematic photography & videography portfolio built with Next.js, GSAP, and Lenis.
Featuring a film reel loading screen, scroll-triggered animations, and smooth page transitions.

## Tech Stack
- Next.js 14 (App Router)
- GSAP 3 + ScrollTrigger + SplitText
- Lenis Smooth Scroll
- Tailwind CSS v4
